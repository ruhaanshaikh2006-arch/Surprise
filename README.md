# I Love You Soo Much Ilma

A small, single-page static website dedicated to Ilma. This repository is ready to be pushed to GitHub and served via GitHub Pages.

## Features
- Heart-shaped photo frames (CSS mask)
- Improved mobile styling
- Animated flower particles (tulip & rose)
- Proposal modal + share functionality
- Clean `/assets` structure (css, js, img)

## How to deploy
1. Clone this repository locally.
2. Commit and push to a GitHub repository.
3. Enable GitHub Pages in repo settings (use `main` branch or `gh-pages` branch), or use the included GitHub Actions workflow to deploy to `gh-pages` branch automatically.

Alternatively, drag-and-drop the folder into Netlify or Vercel for instant hosting.
